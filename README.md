### 🔥 Featured Projects

#### 🏢 [Decentralized Real-Estate Marketplace](https://github.com/muhlikVision/Decentralized-Real-Estate-Marketplace)
A comprehensive Web3 application bridging blockchain and AI. Built with ReactJS and Hardhat, it tokenizes physical properties as ERC-721 NFTs. Features robust Solidity smart contracts for automated escrow, IPFS for decentralized storage, and a Python Flask backend that streams dynamic, machine-learning-driven price predictions.

#### 🛡️ [Scam Checker: Automated Threat Intelligence](https://github.com/muhlikVision/Scam-Checker)
A full-stack rapid-response application featuring a Vite/React frontend and a high-performance Python backend. Integrates the Google Gemini API to analyze user-uploaded media, outputting structured JSON threat assessments. Utilizes SQLAlchemy and PostgreSQL for secure, privacy-compliant logging of scam trends.

#### 🚗 [CARFARE: AI-Integrated Smart Parking](https://github.com/muhlikVision/CARFARE)
An end-to-end cross-platform ecosystem built with Flutter and Dart, ensuring parity across iOS, Android, and Web. Utilizes Firebase Firestore and Cloud Functions for a serverless backend. Integrates Google ML Kit for edge machine vision, enabling real-time optical character recognition (OCR) for automated license plate scanning.
